Factor Risk Attribution Engine

Completed · June 2026

Decomposes daily portfolio returns into systematic (market beta) and idiosyncratic components using rolling 60-day OLS regression on real Nifty 50 constituent data. Computes portfolio-level hedge ratio, HHI concentration index, sector exposure breakdown, and generates five attribution charts. Run on three years of live data: 91% of portfolio return was market-driven, with 1.98% idiosyncratic alpha.

P&L Attribution Tool

Planned · August 2026

Replicates the P&L attribution methodology used in trading risk management: decomposing daily P&L into Delta P&L, Gamma P&L, Vega P&L, Theta decay, and unexplained residual. This is exactly how derivatives desks report performance — breaking down what drove the number so traders and risk managers can understand it. Built in Python with a clean tabular output format.
