Open-source quantitative finance tools built in Python — grounded in methodology I use professionally.

Portfolio Analytics Dashboard

In Progress · June 2026

A Python tool that ingests a portfolio of assets and computes: daily returns, cumulative P&L, Sharpe ratio, Sortino ratio, maximum drawdown, rolling beta, Calmar ratio, and Value at Risk. Built with pandas, matplotlib, and yfinance. The metrics mirror what quant analysts, portfolio managers, and risk teams use daily — not textbook output but production-style analytics.

P&L Attribution Tool

Planned · August 2026

Replicates the P&L attribution methodology used in trading risk management: decomposing daily P&L into Delta P&L, Gamma P&L, Vega P&L, Theta decay, and unexplained residual. This is exactly how derivatives desks report performance — breaking down what drove the number so traders and risk managers can understand it. Built in Python with a clean tabular output format.

Derivatives Pricing Calculator

Planned · September 2026

Black-Scholes option pricing with full Greeks calculator, term structure modelling for interest rates and vol surfaces, and a basic Monte Carlo pricer for path-dependent options. Includes a Streamlit interface so you can interact with the model without running Python directly — change inputs and see how option price and Greeks respond in real time.
